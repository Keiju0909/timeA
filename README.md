# タイムトラッカーアプリ
## プロダクトゴール
##### 目標に向けた行動を継続しやすくする
##### 1日・1週間の目標時間を設定できる
##### 達成率が可視化され、モチベーションを維持しやすい

## プロダクトバックログ
1. ダッシュボード
- 今日の合計作業時間
- 週間合計
- タスク別の割合グラフ（円グラフ or 棒グラフ）

2. タイムトラッキング
- 作業開始 / 停止ボタン - 実装済み
- タスク名の入力 - 実装済み
- 日別・週別の記録表示 - 実装済み

3. 目標設定
- 1日の目標時間
- 週間目標
- 達成率の表示

4. データ管理
- ローカルストレージに保存
- ブラウザを閉じてもデータ保持

## 構成
##### フロントエンドだけで完結する軽量アプリ
##### HTML / CSS / JavaScript のみで構築

## データ保存
##### `timeARecords` に作業記録を保存
##### `timeAActiveSession` に開始中のタスク情報を保存

## 実行方法
##### 1. `index.html` をブラウザで開く
##### 2. もしくは以下のコマンドで簡易サーバーを起動する
```bash
cd /workspaces/timeA
python3 -m http.server 8000
```
##### 3. ブラウザで `http://localhost:8000` を開く
